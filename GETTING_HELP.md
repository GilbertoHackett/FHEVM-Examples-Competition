## üìû How to Report Issues

### Bug Report
1. Check [TROUBLESHOOTING.md](./TROUBLESHOOTING.md)
2. Check [FAQ.md](./FAQ.md)
3. Search existing GitHub issues
4. Create new issue with:
   - Clear title
   - Steps to reproduce
   - Expected behavior
   - Actual behavior
   - Environment info

See [.github/ISSUE_TEMPLATE/bug_report.md](.github/ISSUE_TEMPLATE/bug_report.md)

### Security Issue
1. **DO NOT** create public GitHub issue
2. Read [SECURITY_POLICY.md](./SECURITY_POLICY.md)
3. Email: [security@example.com](mailto:security@example.com)

### Feature Request
1. Check [ROADMAP.md](./ROADMAP.md)
2. Check existing GitHub issues
3. Create new issue with:
   - Clear title
   - Problem description
   - Proposed solution
   - Use case

See [.github/ISSUE_TEMPLATE/feature_request.md](.github/ISSUE_TEMPLATE/feature_request.md)
